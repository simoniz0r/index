Bash Scripts

KDE-gaps

aptmarklist.sh

A script that outputs the number of user installed packages using 'apt-mark'. It also provides arguments to list user installed packages in alphabetical order and to write the output to a file. This script can also install packages from the outputted file.

easyrm.sh

A script that uses 'mv' to move files to '~/.easyrmtmp' and provides arguments to clear out '~/.easyrmtmp'. This script is meant to replace using 'rm' on files and folders in case you aren't sure about deleting them.

lyrics.sh

A simple script for viewing lyrics in your terminal or in a 'zenity' GUI.

nvidiacomposition.sh

A script to remove screen tearing by forcing full composition pipeline on NVIDIA GPUs. Will work with any monitor setup.

speedtest.sh

A simple script that uses 'wget' to download files to '/dev/null' to test download speeds. File sizes available for testing are 5MB, 10MB, 100MB, and 200MB. Also included is a file from Twitch, Steam, and Google.

togglemicmute.sh

A script that can be used to toggle microphone mute/unmute with pulseaudio.

urltotmp.sh

A script that will download a URL from your clipboard to the 'tmp' directory. Originally made for quick picture downloads.

vscodestatusbarcolor.sh

A quick and semi-dirty script that will append the color code of your choice to VSCode's CSS file to change the color of the status bar. Make a backup of your `workbench.main.css` file before using this, and only use it once per update/install.

wttrrefresh.sh

A simple script that runs 'curl --silent "wttr.in/$LOCATION?O?Q'; $LOCATION is asked for when running the script.

youtube-dlclip.sh

A simple script that runs 'youtube-dl' with the URL from 'xclip.'
