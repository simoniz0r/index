apttool

A bash script that provides a simple GUI for running apt options if 'zenity' is installed; run 'apttool gui' to launch with the 'zenity' GUI. It can also be run in a terminal by executing 'apttool'.

gpgpassman

A bash script that uses gpg to encrypt and decrypt passwords stored in '~/.gpgpassman'. Passwords are copied to the clipboard for 45 seconds after decryption. If you have 'zenity' installed, executing 'gpgpassman gui' will launch the 'zenity' GUI. gpgpassman decryption can be ran directly through 'gpgpassman dec' for quick access to your passwords.
