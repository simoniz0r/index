Linux Apps

A few Linux apps made by simonizor. All apps here can be installed using spm

DiscordToolbox

A bash script that installs mods for Discord. Mods that can be installed are BeautifulDiscord, mydiscord, and BetterDiscord.

View Source

gpgpassman

A password manager that uses gpg to encrypt and decrypt passwords stored in ~/.gpgpassman, and gpgpassman has an optional GUI provided by zenity. After decryption, passwords are copied to the clipboard for 45 seconds using xclip.

View Source

kpassman

A password manager that uses gpg to encrypt and decrypt stored passwords and uses kdialog to provide a GUI. Encrypted passwords are stored in ~/.kpassman. After decryption, passwords are copied to the clipboard for 45 seconds using xclip.

View Source

spm

spm is a simple commandline package manager that installs AppImages and precompiled tar archives. spm integrates with AppImageHub to get a list of AppImages, and tar archive information is stored in spm's github repo.

yad-twitch-streamlink

A bash script using 'yad' as a GUI that lists Twitch streams for playback using either 'streamlink' + your favorite player or with just 'mpv' directly.
